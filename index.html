A JavaScript data attributes example by Sevawn Guenther

This page demonstrates how to use JavaScript data attributes to differentiate data.

When you click Show Creature Home, the background color will change color based on their natural environment:

  • land: green
  • water: blue
  • air: yellow

When you click Show Creature Type, the border color and style will changed based on the type of creature:

  • animal: solid red
  • insect: dashed orange

This page functions via the following:

  • document.querySelectorAll() makes a list of the creature elements on this page (each box with a creature, picture, and description in it)
  • .forEach(el => {}) goes through each creature element in this list and performs an action on it
  • the actions being performed are el.style.backgroundColor, which changes the background color of each element, and el.style.border, which changes the border
  • a switch function detects the "home" and "type" of each creature
    • the "home" and "type" are embedded as dataset properties in the creature elements beforehand
  • within the switch, el.dataset.home and el.dataset.creatureType are used to get the "home" and "type" of each creature
  • based on the home and type of each creature, a different background color and border style will be chosen
